# Solicitações de Conexões  

Este menu permite ao usuário da empresa Conciliadora de Cartões solicitar a conexão com o Adquirente de Cartões para recebimento de documentos de seus clientes.  

O tráfego de documentos entre a empresa Conciliadora e a Adquirente de Cartões é realizado pelo sistema EDI quando a Adquirente aprova a solicitação no WebEDI.  

O processo é efetuado da seguinte forma:  

+ O usuário da empresa de Conciliação de Cartões acessa a tela **Cadastro de Solicitações** e preenche o formulário de requisição para conexão com determinado Adquirente para receber os documentos de determinados clientes.  

+ Após a gravação da solicitação, a empresa Adquirente de Cartões deverá acessar a tela **Listagem de Conexões** para conferir se os dados de tipos de documentos estão corretos de cada cliente e efetuar a confirmação/habilitação para iniciar o tráfego desses arquivos entre Adquirente e Conciliadora. Caso os dados estejam incorretos, o usuário da empresa Adquirente poderá recusar a habilitação e informar uma justificativa.  

Acesse os sub menus ao lado para visualizar mais informações de cada item do processo de solicitação.
